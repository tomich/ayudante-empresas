COMANDOS IP (MSDOS)

ipconfig/all

Muestra la configuración de las conexiones de red

ipconfig/displaydns

Muestra la configuración sobre las DNS de la red

ipconfig/flushdns

Borra la caché de las DNS en la red

ipconfig/release

Borra la IP de todas las conexiones de red

ipconfig/renew

Renueva la IP de todas las conexiones de red

ipconfig/registerdns

Refresca DHCP y registra de nuevo las DNS

ipconfig/showclassid

Muestra información de la clase DCHP

ipconfig/setclassid

Cambia/modifica el ID de la clase DHCP
